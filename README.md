# Flutter Map Location Application

โปรเจกต์นี้เป็นแอปพลิเคชันที่พัฒนาด้วย Flutter  
มีวัตถุประสงค์เพื่อแสดงแผนที่และตำแหน่งปัจจุบันของผู้ใช้งาน  
โดยใช้แพ็กเกจ `flutter_map` ร่วมกับ OpenStreetMap

## ความสามารถของระบบ
- แสดงแผนที่จาก OpenStreetMap
- ค้นหาตำแหน่งปัจจุบันของผู้ใช้งานด้วย Geolocator
- แสดงตำแหน่งผู้ใช้บนแผนที่ด้วย Marker
- แสดงวงกลมรัศมีความคลาดเคลื่อน (Accuracy) รอบตำแหน่งผู้ใช้
- สามารถเปลี่ยนรูปแบบแผนที่ได้ (แผนที่ปกติ / แผนที่โทนมืด)

## วิธีการใช้งาน
1. เปิดแอปพลิเคชัน
2. กดปุ่มค้นหาตำแหน่ง เพื่อแสดงตำแหน่งปัจจุบันของผู้ใช้
3. ระบบจะแสดง Marker และวงกลมแสดงค่า Accuracy บนแผนที่
4. กดปุ่มเปลี่ยนเลเยอร์ เพื่อสลับรูปแบบการแสดงผลของแผนที่
